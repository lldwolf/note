第N章 邮件存储设置

在 工具>选项>维护 选项卡中，有个 存储文件夹 的按钮，点一下选择 更改将你现在的邮件移动到别的非系统分区上的一个目录，等你安装好了，再做个同样的动作，把邮件目录指向这个备份的目录，就会导出来了。

例如，我的机器上的默认备份目录为C:\Documents and Settings\Administrator\Local Settings\Application Data\Identities\{393C8B50-DABE-4E23-A7CE-DF7EF28006E4}\Microsoft\Outlook Express

创建新的数据文件

  1. 创建新目录并设为 Default 目录

  2. 改变默认本地目录，该设置重启后生效

  3. 重新定义所有 Rule 的转发目录

  4. 重新创建所有的 Search Folder
