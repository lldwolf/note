第N章 快捷键

主窗口

快捷键 说明
Ctrl + 1 切换至 Mail
Ctrl + 2 切换至 Calendar
Ctrl + 6 切换至 Folder List, 在该模式下很像 Mail, 但是 Favorites 将会消失
Ctrl + F Forward 邮件
Ctrl + R Reply 邮件
Ctrl + Shift + R Reply All 邮件
F6 在 Outlook 的各个面板间切换

写邮件

快捷键 说明
Ctrl + K 检查姓名
Alt + > 打开地址薄并将选中的人加入收件人列表
Ctrl + Shift + >/< 在写邮件时放大/缩小字体
Ctrl + B 粗体切换
Ctrl + I 斜体切换
Ctrl + U 下划线切换
Alt + S 发送邮件
Alt + HIN 去掉背景色
Alt + NAF 添加附件
