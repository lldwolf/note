ORACLE版的特殊配置

需要修改下列注册表项

  • HKEY_LOCAL_MACHINE\SOFTWARE\ORACLE\NLS_LANG
  • HKEY_LOCAL_MACHINE\SOFTWARE\ORACLE\HOME0\NLS_LANG

若ORACLE字符集为中文，则设为SIMPLIFIED CHINESE_CHINA.ZHS16GBK，若为英文，则设为AMERICAN_AMERICA.WE8ISO8859P1，英文可能会有不同，可用下列SQL语句查看

select userenv('language') from dual
