第N章 Spec Binder

用于打包规格书，可批量打印。

N.1 新建

例如，新建流量类仪表包

运行菜单【Actions】-【New Technical Specification】，输入相应项，如下图所示

在新建好的Flow节点下的Specification Sheets节点上使用鼠标右键菜单【Assign Specification】，选择需要的位号加入即可。

N.2 删除

在删除前，需要先删除所有规格书，在Specification Sheets节点下的每个位号节点上使用右键菜单【Remove Specification】，删除规格书。所有规格书删除完毕后，可将Technical Specification节点删除。
