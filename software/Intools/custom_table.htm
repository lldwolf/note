第N章 Custom Table

INtools中定义了16张表，用于用户扩充一些数据。说也是说不明白，我直接举个例子说明一下用途好了。

第一步，首先在Domain Administrator中激活自定义表，菜单 Activities - Custom Tables, 我选中 DEMO 域的 New Refinery 装置，激活第一张自定义表，我给它起名为creator，用于存放创建者，界面如下图所示：

第二步，我们就可以在INtools中使用了，在Index模块中察看任意一个位号的属性，你会发现属性窗口中的 Custom Table 也可用了，点击 creator 后的新增数据的小按钮，随便录入一些测试数据，界面如下图所示：
