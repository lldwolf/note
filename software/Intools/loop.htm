第N章 回路图

N.1 定义输出AutoCAD图纸

N.1.1 路径设置

INTOOLS 6.0支持的最高AutoCAD版本为2004。

在 File -- Preferences -- Loop Drawing -- CAD File Location里面定义5个路径：

项 说明
CAD folder AutoCAD安装路径
CAD configuration folder AutoCAD安装路径
CAD function folder INTOOLS里面的CAD函数目录，默认为%INTOOLS%\CadFunc
Open drawing files from INTOOLS里面的CAD图纸目录，默认为%INTOOLS%\DWG
Drawing block folder INTOOLS里面的CAD块目录，默认为%INTOOLS%\Blocks

N.1.2 设定块类型

回路图模块下方Block Types按钮，新增的类型其General属性为no.

N.1.3 设定块

即图纸文件，回路图模块下方Blocks按钮。在这里设定需要用到的图纸文件。

N.1.4 设定宏

即要生成的数据，回路图模块下方Macro Def.按钮，定义宏的名称及其对应的数据库字段值。

N.2 常见问题

N.2.1

问题描述：已完整接线，却无法查看回路图。

问题原因：常见于带屏蔽线的接线情况，在接线时，现场设备并不接屏蔽线，屏蔽线是从接线箱开始的，如果查看屏蔽线的接线路径，会发现屏蔽线的接线路径不是从1开始，而是从最末端的100倒排回去，在接线界面中双击电缆芯颜色块，将屏蔽线的Signal Level(表示接线中的第几行) 和Sequence（表示在回路中的第几段）作相应的修改即可。
