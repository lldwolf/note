N.1 ISA 命名规则

命名规则的每一段由两部分组成：分隔符和描述符。你可以使用任何字符作为分隔符，不过，我们伟大光荣正确的Intergraph预定义了一套分隔符，您老可以在下拉列表框中选择

分隔符出现在每一段的前面，例如在仪表位号的命名规则中，第一个分隔符为‘U’，其它的分隔符为‘-’或‘/’，您也可以不设，以免碍眼。

前缀（第一段）一般用来表示位号所在的DPAU。

N.3 Flexible 命名规则

分段，但你可以设定下列任意内容作为一段：

段 描述
AREA NAME 区域名
COMPONENT NUMBER 位号的数字段，例如位号 101-FT-202/A，数字段为202
EQUIPMENT 设备
FREE SEGMENT 任意字符串
INSTRUMENT TYPE 仪表类型

因为没有强制的仪表类型段了，所以在新建位号时，必须要手工关联其仪表类型。
LINE Line Number
LOOP FUNCTION 回路功能
LOOP MEASURED VARIABLE 回路的测量变量
P&ID P&ID 图号
PLANT NAME 装置名
SUFFIX 位号或回路号后缀

后缀用于批量创建回路或位号（如何批量创建位号）
UNIT NAME 单元名
UNIT NUMBER Unit Number

N.4 Free Format 命名规则

只有一段，你想咋写就咋写。

注意：因为没有强制的仪表类型段了，所以在新建位号时，必须要手工关联其仪表类型。
