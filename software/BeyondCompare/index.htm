Beyond Compare

在比较目录时，过滤不需要比较的文件

列如，忽略 CVS, bin, obj 目录，以及所有的 .bak 文件

Seesion -> Session Settings -> Nmae Filters

上述配置将会出现在工具栏的 Filters 中，大体格式如下：

-*.class;-*.bak;-*.licx;-.cvsignore;-CVS\;-obj\;-bin\;-Assembly\;-Publish\;-test\

只显示不同的文件

不比较空白字符

放弃默认的快速比较方式，改用其于内容的比较

Seesion -> Session Settings -> Comparison

导出比较结果（文件列表）

点一下 "Expand All" 工具栏按钮展开全部，使用 “Session -> Folder Compare Report"

常用快捷键

可通过 Tools - Customized Commands 自定义快捷键

Shortcut Description
Ctrl + L Copy to Left
Ctrl + R Copy to Right
Ctrl + P Previous difference
Ctrl + N Next difference
