第N章 常用快捷键

快捷键 说明
Ctrl + N 新建网页
Ctrl + U 打开首选参数设置
F4 关闭所有面板
Ctrl + F3 打开属性面板
Ctrl + Shift + > 缩进选中代码
Ctrl + Shift + < 凸出选中代码
Ctrl + Alt + I 插入图片
Ctrl + Alt + T 插入表格
Ctrl + M 在表格中插入行
Ctrl + Shift + M 在表格中删除行
Ctrl + Shift + A 在表格中插入列
Ctrl + Shift + - 在表格中删除列
Ctrl + Shieft + A 在网页中插入命名锚记
Ctrl + L 插入超链接
Ctrl + T 插入HTML代码
