N.2 直线

N.2.1 创建直线

下例创建一条从(0, 0)到(300, 300)的直线

Dim oLine As AcadLine
Dim Pt1(0 To 2) As Double, Pt2(0 To 2) As Double

Pt1(0) = 0: Pt1(1) = 0: Pt1(2) = 0
Pt2(0) = 200: Pt2(1) = 200: Pt2(2) = 0
Set oLine = ThisDrawing.ModelSpace.AddLine(Pt1, Pt2)

N.2.2 线型

N.2.2.1 在AutoCAD中加载

命令: linetype

在弹出窗口中按“加载”按钮，即加加载所需的线型

N.2.2.2 使用VBA加载

例如，要加载DASHED2线型，VBA代码如下：

Dim entry As AcadLineType
Dim found As Boolean
found = False
For Each entry In ThisDrawing.Linetypes
    If StrComp(entry.Name, "DASHED2", 1) = 0 Then
        found = True
        Exit For
    End If
Next
If Not (found) Then ThisDrawing.Linetypes.Load "DASHED2", "acadiso.lin"

Set oLine = ThisDrawing.ModelSpace.AddLine(Pt1, Pt2)
oLine.Linetype = "DASHED2"
