第N章 使用窗体控件

N.1 CheckBox

使用控件工具箱工具面板。放置好CheckBox后，点击面板上的可切换设计模式，在设计模式下可更改它的属性及位置，在非设计模式下点击CheckBox即可改变它的值。

使用VBA添加CheckBox的代码如下：

Sub InsertChk()

    Dim r As Range
    Set r = ActiveSheet.Range("C5")
    Dim chk As OLEObject
    
    Set chk = ActiveSheet.OLEObjects.Add(ClassType:="Forms.CheckBox.1", Link:=False, _
        DisplayAsIcon:=False, Left:=r.Left, Top:=r.Top, Width:=108, Height:= _
        19.5)

    ActiveSheet.OLEObjects(chk.Name).Object.Caption = "test"         '显示值
    ActiveSheet.OLEObjects(chk.Name).Object.SpecialEffect = 0        '外观
    ActiveSheet.OLEObjects(chk.Name).Object.AutoSize = True          '自动调整大小
    ActiveSheet.OLEObjects(chk.Name).Object.Value = True             '设为选中状态

End Sub
