N.1 字体

获取方式为 Excel.Range.Font，下表展示了与字体有关的设置(全部是Font的子属性)

属性 说明
Bold 是否粗体
Color 颜色，可通过RGB函数生成
ColorIndex 颜色索引号
Italic 是否斜体
Name 字体名称
Size 字体大小
Strikethrough 是否有删除线
Subscript 是否下标
Superscript 是否上标
Underline 下划线设置，设置值为Excel.XlUnderlineStyle，可设无下划线、单下划线、双下划线、会计用单下划线、会计用双下划线

N.2 对齐

下表展示了与对齐有关的属性（Excel.Range的子属性）

属性 说明
HorizontalAlignment 水平对齐方式，值类型为Excel.XlHAlign
IndentLevel 缩进值
MergeCells 是否自动合并单元格
Orientation 旋转角度，单位为角度
ReadingOrder 文字方向
ShrinkToFit 是否缩小字体填充
VerticalAlignment 垂直对齐方式，值类型为Excel.XlVAlign
WrapText 是否自动换行

N.3 数字

使用Excel.Range.NumberFormatLocal属性，具体的格式字符串可以到Excel中进行测试，最常用的一个是将单元格设为文本模式，如下所示

Sheet1.Range("A1").NumberFormatLocal = "@"

N.4 边框

通过Range.Borders获取边框线，这是一个集合，通过Excel.XlBordersIndex枚举来索引各个边，对每条边框线，有如下属性可设

属性 说明
ColorIndex 颜色索引值
LineStyle 线型，其类型为Excel.XlLineStyle
Weight 线宽，其类型为Excel.XlBorderWeight

N.5 图案

通过Range.Interior属性，对其有如下属性可设置

属性 说明
ColorIndex 填充颜色索引值
Pattern 填充模式，常用的值为xlSolid，其它值自己到Excel中去录制吧
PatternColorIndex 填充模式的颜色索引，一般设为xlAutomatic

N.6 保护

属性 说明
FormulaHidden 是否隐藏
Locked 是否保护
