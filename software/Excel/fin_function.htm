第N章 金融函数

IRR 收益率计算

返回由数值代表的一组现金流的内部收益率。这些现金流不必为均衡的，但作为年金，它们必须按固定的间隔产生，如按月或按年。内部收益率为投资的回收利率，其中包含定期支付（负值）和定期收入（正值）。

语法

IRR(values, [guess])

基本上，guess 参数可以忽略，Values 为数组或单元格的引用，这些单元格包含用来计算内部收益率的数字

  • Values 必须包含至少一个正值和一个负值，以计算返回的内部收益率。
  • 函数 IRR 根据数值的顺序来解释现金流的顺序。故应确定按需要的顺序输入了支付和收入的数值。
  • 如果数组或引用包含文本、逻辑值或空白单元格，这些数值将被忽略。

RATE 利率计算

返回年金的各期利率。 RATE 通过迭代法计算得出，并且可能无解或有多个解。如果在进行 20 次迭代计算后， RATE 的相邻两次结果没有收敛于 0.0000001， RATE 将返回错误值 #NUM!。

语法

RATE(nper, pmt,pv, fv, type, [guess] )

参数说明

参数 说明 备注
nper 年金的总付款周期数。例如，如果您获得一笔 4 年期的汽车贷款，并按月分期付款，您的贷款有 4*12（或 48）个周期。您需要在公式中输入 48 作为 nper。  
pmt 每个周期的付款。付款不会随着年金的长短发生变化。通常情况下， pmt 包含本金及利息，并且没有其他税费。例如，一笔利率为 12％ 的 10,000 美元 4 年期的汽车贷款的月付款为 263.33 美元。您可以在公式中输入 -263.33 作为 pmt。 如果 pmt 被忽略，则必须包括 fv 参数
pv 现值 ― 一系列未来付款当前值的累积总和  
fv 未来值，或您在进行最后一次付款后希望得到的现金余额。例如，如果您节约 50,000 美元用于在未来 18 年中支付某特殊项目，则 50,000 美元即是未来值。随后，您可以保守估计利率情况并确定每个月必须节约多少钱。 如果 fv 被忽略，则假定为 0
type 数字 0（零）或 1 并指明付款到期日
  • 0（零） = 付款在期末
  • 1 = 付款在期初

年金是在一段连续时间内支付的一系列恒定现金付款，例如，汽车贷款或抵押。在年金函数中，您支付的现金（如储蓄存款）由负数表示；您收到的现金（如股息支票）由正数表示。例如，如果您是存款人，则银行的 1,000 美元存款将由参数 -1000 表示；如果您是银行，则由参数 1000 表示。

PMT 等额分期付款额计算

基于等额分期付款方式及固定利率，计算贷款的每期付款额。

语法

PMT(rate,nper,pv, fv, type )

参数说明

参数 说明 备注
rate 每个期间的利率。例如，如果您以 10％ 的年利率获得一笔汽车贷款，并按月分期付款，您每月的利率为 10％ 除以 12，即 ％0.83。您可以在公式中输入 10%/12、0.83% 或 0.0083 作为 rate。 请确保用于指定 rate 和 nper 的单位保持一致。如果偿还期为 4 年且年利率为 12% 的月还款，则使用 12%/12 作为 rate，4*12 作为 nper。如果按年偿还相同的贷款额，则使用 12% 作为 rate，使用 4 作为 nper。
nper 年金的总付款周期数。例如，如果您获得一笔 4 年期的汽车贷款，并按月分期付款，您的贷款有 4*12（或 48）个周期。您需要在公式中输入 48 作为 nper。 请确保用于指定 rate 和 nper 的单位保持一致。如果偿还期为 4 年且年利率为 12% 的月还款，则使用 12%/12 作为 rate，4*12 作为 nper。如果按年偿还相同的贷款额，则使用 12% 作为 rate，使用 4 作为 nper。
pv 现值，或一系列未来付款的当前值的累积和，也称为本金。  
fv 未来值，或您在进行最后一次付款后希望得到的现金余额。例如，如果您节约 50,000 美元用于在未来 18 年中支付某特殊项目，则 50,000 美元即是未来值。随后，您可以保守估计利率情况并确定每个月必须节约多少钱。 如果 fv 被忽略，则假定为 0（例如，贷款的未来值为 0）。
type 数字 0（零）或 1 并指明付款到期日
  • 0（零） = 付款在期末
  • 1 = 付款在期初

PV 现值计算

返回投资的现值。现值是一系列未来支出现在所值的总额。例如，您在贷款时，贷款额就是支付给贷款人的现值。

示例

假设您正在考虑购买一项保险年金，年金的购买成本为 $60,000，该年金每月月底支付您 $500 ，付款年限为 20 年。，而您的预期投资收益率为 8%。您需要确定这是否是一项好的投资。

结果为负值，因为这是一笔付款，亦即支出现金流。如果年金的购买成本是 (60,000)，则年金的现值 (59,777.15) 小于实际支付值。因此，这不是一项合算的投资。
