Disable FxCop Rules

对于 SonarQube 4.5 以上版本，默认集成了 FxCop Rules，但需要 MSBuild 14 支持。

对于老的项目，可使用以下方式禁用所有的 FxCop rules

  1. login SonarQube with admin access
  2. Quality Profiles → C# Profiles → Sonar way
  3. Choose FxCop Repository
  4. Disable all FxCop rules in the searching result

Other Rules

I also update some sonar way rules according to my project.

Rule Change
"value" parameters should be used I updated its severity from Critical to Minor. Sometimes I need a property for data-binding field which is a popup window.
Method name should comply with a naming convention I updated the format to "^[A-Z][a-zA-Z0-9_]*[a-zA-Z0-9]$" because the default auto-generated event method name doesn't comply the default format.
Source files should have a sufficient density of comment lines I updated its severity from Majo to Info
Multiline blocks should be enclosed in curly braces I updated its severity from Critical to Majo
