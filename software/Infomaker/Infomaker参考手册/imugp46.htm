报表选项

本节讲述如何设置报表的一些默认选项，例如颜色、边框等。

这些选项都是用于行式结构的报表，包括Freeform、Grid、Label、N-Up、Tabular、Group、Crosstab。InfoMaker为每种样式设了不同的选项。

当创建报表时，向导会提示你设置显示选项。

设定显示选项的步骤如下：

  1. 选择菜单项 Design>Options，打开报表选项设置对话框。

  2. 选择Generation面板。

  3. 在Presentation Style下拉列表框中选择要修改的报表样式，一开始各显示的是该样式当前使用的设置。

  4. 下面是对各个选项的解释：

    选项 含义
    Background color 默认的背景色
    Text border and color 默认的边框颜色（用于卡片和标题）
    Column border and color 显示数据时的边框颜色
    Wrap Height (仅用于Freeform) 条目栏的高度当值为空时（即不考虑值的长度带来的影响），报表使用的列数决定义条目栏的高度。各列显示在同一栏内。当设定该选项的值后，条目栏的行高会定被设为给定值。

    颜色选择 不需翻译。If you select Window Background, Application Workspace, Button Face, or Window Text from the Color dropdown listbox, the report will use the colors specified for the windowing environment in which the report is running.

  5. 点OK。


保存设置

InfoMaker会保存你的设置为创建相应报表时的默认选项。
