使用报表向导

报表向导

报表向导可以帮你创建不同样式的报表。

使用报表向导

  1. 点击工具栏上的New按钮

  2. 在New对话框中，选择Object面板。



  3. 选择需要的报表样式图标，点OK。


报表向导的功能

下面是报表向导的功能：

Using this report wizard You create a new report
Composite 可以将多个报表合成一个报表
Crosstab 带有汇总数据，样式类似于spreadsheet表格。
Freeform 前面一个Label（一般为说明文字），后面一个数据列，从上往下排列
Graph 以图表方式显示数据
Grid 以行列方式显示数据
Group 可对数据进行分组显示
Label 以卡片方式显示数据
N-Up 每个数据栏包含多行数据，请参考示例，这里说不清楚
OLE 2.0 一个OLE对象
RichText 将多个数据库格式化为文本
Tabular 数据列占满全页，并且每列头都有标题

报表向导能够生成To-Do List，有关To-Do List，请参考"Using the To-Do List".

有关报表向导，请参考第4章 "定义报表"。
