Appendix A Adding Functions to the PBODB80 Initialization File

About this appendix

In general, you should not need to modify the PBODB80 initialization file . In certain situations, however, you may need to add functions to the PBODB80 initialization file for connections to your backend DBMS through the ODBC interface in InfoMaker.

This appendix describes how to add functions to the PRODB80 initialization file if necessary.
