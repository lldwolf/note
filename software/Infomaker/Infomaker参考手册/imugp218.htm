PageCountAcross

Description

Gets the total number of horizontal pages when viewing a DataWindow object or report (in Print preview) that is wider than the Print Preview window.

Syntax

PageCountAcross ( )

Return Values

Long. Returns the total number of horizontal pages if it succeeds and -1 if an error occurs.

Usage

PageCountAcross applies to Print Preview.

Examples

This expression returns the number of horizontal pages in the Print Preview window:

PageCountAcross ()
