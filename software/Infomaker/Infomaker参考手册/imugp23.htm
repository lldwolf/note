Library 简介

InfoMaker使用Library来控制对象，一个Library是一个扩展名为“PBL”的文件：

filename.pbl

在InfoMaker中，用户使用Library Painter在Library中操作各种对象。

Library painter 的功能

The Library painter workspace displays objects in all libraries on your computer, but you can set the Library painter to display only objects in the current library. In the Library painter, you can:

  • 打开并编辑当前Library中的对象。
  • 在Library间复制、移动或删除对象。
  • Migrate, rebuild, and regenerate libraries（@_@，rebuild和regenerate真难翻啊）
  • 在应用程序中绑定报表、Form和Pipeline，有关应用程序的知识请参考第17章 "使用Application".

Library painter 不能干的事

不能创建Library，也不能给Library重命名，有关创建Library的知识请参考"新建Library".

不能新建报表、Form、Query、Pipeline。

不能打开不属于当前Library的对象。
