选择报表样式

The presentation style you select for a report determines the format InfoMaker uses to first display the report in the Design view. You can use the format as displayed or modify it to meet your needs:

你可以在新建报表的对话框中选择以下样式：

  • Tabular
  • Freeform
  • Grid
  • Label
  • N-Up
  • Group
  • Composite
  • Graph
  • Crosstab
  • OLE 2.0
  • RichText

N-Up 样式

真不知道该怎么形容这种样式，简单地讲，它可以把一个数据集平均分成多栏显示，例如如果一个数据集大家可以想象一个数据集只有1列，那么打印起来是比较浪费的，使用N-Up样式，我们可以将这些数据平均分布在6列上，也就是说，把这一列数据集平均分成了6列来显示。利用这种特性用户可以定制一些数据带有周期性的报表，例如值日表。

选完数据源后，InfoMaker会要求你设置要分的栏数

对数据源中的每列数据，InfoMaker根据你设定的栏数n生成n栏（column_1 至 column_n）。
