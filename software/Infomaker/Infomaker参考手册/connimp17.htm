This chapter gives general information about using the OLE DB interface. For more detailed information:

  • Use the online Help provided by the data provider vendor.
  • Check to see if there is a technical document that describes how to connect to your OLE DB data provider. Any updated information about connectivity issues is available from the Sybase Customer Service and Support Web site .
