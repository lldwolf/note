从命令行启动InfoMaker

InfoMaker能够从命令行启动，启动后还能顺便打开下列Painter或工具之一：

Database painter Library painter
Data Pipeline painter Query painter
File Editor Report painter
Form painter

命令行语法如下： 

{ win } directory\im80.exe /P paintername

参数说明：

参数 描述
directory InfoMaker的路径，不过一般情况下它在系统的Path变量中，所以不用输入路径也行。
paintername 要打开的Painter的名字。这个名字得能唯一地标识出一个Painter。也不必输入完整的名字，例如，输入“q”即可打开Query Painter，输入“datab”即可打开Database Painter。如果输入完整名称，则要去掉里面的空格，例如“DataPipeline”而不是“Data Pipeline”。大小写不敏感 。如果要打开文件编辑器，可使用参数“FI”或“fileeditor”。

打开对象或创建新对象

You can also add one or more of the following optional switches to the command line to open a specific object or create a new one. 

{/L libraryname} {/O objectname} {/N} {/R} {/RO} {/A arguments}

All of these switches must follow /P paintername , as shown in the examples after the tables.

Switch Description
/L Identifies the library that contains the object you want to open
/O Identifies the object, such as a report, that you want to open
/N Creates a new report
/R Runs the report specified with /O and allows designing
/RO Runs the report specified with /O but does not allow designing
/A Provides retrieval arguments for the report specified with /O
Parameter Description
libraryname The name of the library that contains the object you want to open. The default is the library specified in the DefLib variable in the [PB] section of the InfoMaker initialization file
objectname The name of the object you want to open
arguments For a report, retrieval arguments for the specified report. Arguments must be in the correct order, separated by semicolons (;). Array argument values must be separated by commas (,). Decimal arrays are not supported

Examples

The following examples use im8 to represent the directory where InfoMaker is installed.

Enter this command at the Windows MS-DOS prompt to start InfoMaker and open the Database painter:

win im8\im80.exe /P datab

Enter this command in the Windows Run dialog box to start InfoMaker and open the report called d_emp_report in the library master.pbl:

im8\im80.exe /P report /L master.pbl /O d_emp_report
