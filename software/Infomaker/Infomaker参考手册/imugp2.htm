关于InfoMaker

InfoMaker 是一个报表工具(注：而且是很难用的)

InfoMaker 是一个强大易用的报表工具（注：当然要这样宣传了，相不相信是你的事），你可以使用它查询数据库，并且以用户自定义的方式显示这些数据。

不过InfoMaker并不仅仅是一个单纯的报表工具。如果安装了Panters插件，它还可以直接操作数据库的内容。

InfoMaker 是一个数据助手（注：如果我有这么一个助手，我会立刻把它开除）

一般情况下，你不需要懂得编程，就可以使用InfoMaker以多种方式来处理数据（注：每一种方式都是很难用的，所以没必要费神选哪种方式合适）。

InfoMaker提供内置的数据库连接方式，可以连接各种数据库，包括桌面数据库和网络数据库（注：除了SYBASE数据库是自己的，别的就是ODBC，如果你正在做不装ORACLE客户端就能通过InfoMaker来连数据库的美梦，请赶快醒一醒）。某些版本的InfoMaker还包括强大的Adpative Server Anywhere数据库的管理系统，你可以用来创建数据库，使用内置的EAS数据库（一个样本数据库）来创建报表或其它的InfoMaker对象（注：还有人用这种桌面数据库吗？）。

有关DBMS的信息，参考连接你的数据库.

当你使用InfoMaker时，你使用的是图形界面，也就是说你不需要懂SQL语句就可以处理数据库的数据（注：如果你懂SQL，那可不在我们的计划之内，请把它忘掉后再使用InfoMaker）。InfoMaker能够根据你的报表和其它的图形对象在后台自动创建好SQL语句（注：至于创建的SQL语句的效率，你可不要跟我讨论，反正你是不懂的，相信我创建的是最好的就行了）。

你能够使用InfoMaker干啥？

你可以使用InfoMaker创建下面的东东：

  • 用于浏览数据的报表
  • 浏览或修改数据的表单
  • 查询（Query）：自动从数据库中取出数据填到报表或表单中（注：如果你不懂SQL，你不知道怎么建，如果你懂SQL，你还是不知道怎么建）
  • 数据通道（Pipeline）：把数据从一个数据库导到另一个数据库中。
  • 应用程序：把报表和表单打包并分发给用户。

You can see many examples of the objects you create in InfoMaker in this chapter.

Painters

In InfoMaker, you do your work in painters. A painter is an object editor you use to create and work with objects of a particular type. For example, in the Report painter, you create and work with reports, and in the Data Pipeline painter, you create and work with data pipelines.

If you do not see all the available painters When you installed InfoMaker, you (or others in your organization who install and set up your software) chose various combinations of painters to install. You can add or remove any component, except for the base, at any time. The base combination includes the Report painter, the Query painter, and the Library painter. Additional optional painters include the Database painter, the Form painter, and the Data Pipeline painter.

For complete information about installing InfoMaker, see the Installation Guide .
