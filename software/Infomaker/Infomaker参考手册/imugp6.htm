使用对象

在InfoMaker中，你可以：

  • 创建新对象
  • 打开已存在对象
  • 运行或预览对象

创建或打开对象后，对象显示在它的Painter中，你可以在Painter中操作它。

创建新对象

要创建新对象，步骤如下：

  1. 点击工具栏上的New按钮

    或

    菜单中的File>New

  2. 在新建对话框中，在Object面板中选择想创建的对象类型。

    Object 面板内容如下：



  3. 选择一个图标并点OK

    If you chose the Object tab page and you are creating a report, at this point you use a wizard. If you chose the Database tab page, you can create a query or a data pipeline and the new object opens in the appropriate painter.


可用的对象

新建对话框有4个面板，2个是用于创建新对象的：

面板 内容
Object 表单和报表向导
Database 创建查询和数据通道

打开已存在对象

To open existing objects, you use the Open button in the PowerBar.

To open existing objects:

  1. Click the Open button in the PowerBar.

    or

    Select File>Open from the menu bar.

  2. In the Open dialog box, select the object type from the Object Type dropdown listbox and then the object you want to open.



  3. Click OK.

    The object opens in the appropriate painter.


使用最近打开过的对象

You can quickly open recently opened objects by selecting File>Recent Objects from the menu bar. The Recent Objects list includes the eight most-recently-opened objects, but you can include up to 36 objects on the list.

To modify the number of recent objects:

  1. Select Tools>System Options from the menu bar.

  2. In the System Options dialog box (General tab page), modify the number for the recent objects list.


运行或预览对象

To run a form or preview a report, use the Preview button in the PowerBar.

To run or preview an object:

  1. Click the Preview button in the PowerBar.

    or

    Select File>Run/Preview from the menu bar.

  2. In the Run/Preview dialog box, select the object type from the Object Type dropdown listbox and then the object you want to run or preview.



  3. Click OK.

    The object runs or is previewed.
