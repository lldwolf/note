Cos

Description

Calculates the cosine of an angle.

Syntax

Cos ( n )
Argument Description
n The angle (in radians) for which you want the cosine

Return Values

Double. Returns the cosine of n.

Examples

This expression returns 1:

Cos (0)

This expression returns .540302:

Cos (1)

This expression returns - 1:

Cos (Pi(1))
