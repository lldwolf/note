第二章 Eclipse的安装及配置

2.1 安装

严格来说，Eclipse并不需要安装，你只需要到 http://www.eclipse.org 上下载一个Eclipse的最新版本并在本地解压即可。

2.2 系统参数配置

要运行Eclipse，需要JDK的支持，JDK的最新版本可从 http://java.sun.com 上得到。安装完JDK后，需添加或修改如下几个环境变量：

  • JAVA_HOME：JDK安装后的根目录，如 D:\java\jdk1.5
  • CLASSPATH：一些常用的JAVA类库路径，为 %JAVA_HOME%\lib
  • PATH：在最前面加入 %JAVA_HOME%\bin，之所以加在最前面是因为有些软件，例如ORACLE或VisiBroker等会加入自己的JRE路径，而操作系统在搜索时是按顺序来，加在最后有可能会使Eclipse找不到正确的程序。

启动问题解决

与 JDK 版本兼容

如果是 32 位的 Eclipse, 则需要使用 32 位的 JDK 启动, 反之亦然, 否则会在启动时报错

failed to load the jni shared library

可通过修改 eclipse.ini 来实现, 如下所示(位置很重要)

-vm
C:\Program Files (x86)\Java\jdk1.8.0_45\bin\javaw.exe
-vmargs
-Dosgi.requiredJavaVersion=1.7
