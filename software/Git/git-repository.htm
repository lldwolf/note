新建远程库

GIT 是非集中式的版本控制系统，因此没有CVS或SVN那样的服务器。因此有如下特点：

  • 任何一个结点都可以作为服务器，也可以作为客户端
  • 可以将任何目录建为 Repository

可仿照如下步骤建立 Repository

# 登录 lnycrecmoap1d.nam.nsroot.net
$mkdir git_test
$cd git_test
		
# 初始化 git 环境, 使用 bare 模式 
$git init --bare

# 将当前目录加为 Git Repository
# $git add .

然后就可以在本地用类似的命令 clone 了

Server 端设置

  1. 需要开放 GITRoot 写权限

    $chmod -R 775 /opt/gollf/GITRoot

  2. 将 repository 设为 bare 模式

    $cat /opt/gollf/GITRoot/gollf/.git/config

[core]
        repositoryformatversion = 0
        filemode = true
        bare = true
        logallrefupdates = true

  3. 加用户权限

    $cat /opt/gollf/GITRoot/gollf/.git/config

[user]
        name = xxxxx
        name = ooooo
