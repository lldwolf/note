检查字符串格式

1. 检查字符串是否为数字串

所用语言: javascript

检查是否为数字串

function CheckInteger(str){
	var pat = new RegExp("^[0-9]+$");
	if(str.match(pat) == null)
		return false;
	else
		return true;
}

2. 检查字符串是否为整数（包括正负）

var pat = new RegExp("^[+|-]{0,1}[0-9]+$");

3. 检查字符串是否为浮点数（不包括前面的正负号）

var pat = new RegExp("^[0-9]+([.]{0,1}[0-9]+)$");
