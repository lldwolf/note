第N章 最大公约数和最小公倍数

N.1 欧几里德算法描述

设两数为ab，a＞b：

  1. a除以b得余数c；
  2. 令a=c，b除以a得余数c；
  3. 如果a不等于b则跳回1)，否则结束，最后一次的余数就是两数的最大公约数。

得出最大公约数m后，最小公倍数为 a * b / m

private static int GetMaxCommonDivisor(int i1, int i2)
{
    int i3;
    
    if(i1 < i2)

        i3 = i1;
        i1 = i2;

        i1 = i2;
        i2 = i3;

}
