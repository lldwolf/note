Java 练习题 - 读取系统属性

要求

读取相关的系统变量, 例如当前路径, 根目录, 用户名等

示例代码

private static final String FILE_SEPERATOR = System.getProperty("file.separator");

系统变量列表

变量名 说明
file.separator 文件夹分隔符, Windows 为 \, Linux 为 /
user.dir 当前项目的根目录
os.name 操作系统名称
user.language 当前操作系统语言
java.class.path Class Path
line.separator 行分隔符, Windows 为 \r\n, Linux 为 \n
