打包时如何打包 properties 或 XML 配置文件

推荐将这些文件放在 src/main/resources 目录下，这样 Maven 会自动打包

或者在 pom.xml 中配置如下

<build>
    <resources>
        <resource>
            <directory>src/main/java</directory>
            <includes>                      
                <include>**/*.xml</include>
            </includes>
        </resource>
        <resource>
            <directory>src/main/resources</directory>
        </resource>
    </resources>
</build>

如何跳过 test 阶段

mvn clean package -Dmaven.test.skip=true
