Java 练习题 - 遍历 Map

要求

遍历 Map 的所有元素, 打印 key 及 Value

示例代码

Map<Integer, String> map = new HashMap<Integer, String>();

for (int i = 1; i < 10; i++) {
	map.put(i, "Val" + i);
}

Iterator<Entry<Integer, String>> iterator = map.entrySet().iterator();

while (iterator.hasNext()) {
	Entry<Integer, String> currentEntry = iterator.next();
	System.out.println(String.format("%d: %s", currentEntry.getKey(), currentEntry.getValue()));			
}
