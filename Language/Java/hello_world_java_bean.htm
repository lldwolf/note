HelloWorld――Java Bean

使用一个Java Bean，其中一个属性为Name，例如，其Name值为Peter, 则显示“Hello Peter!”

关键代码如下：

Bean 类（PeopleBean.java）

package lld.test.helloworld;

public class PeopleBean
{
	private String name;

	public String getName()
	{
		return name;
	}

	public void setName(String name)
	{
		this.name = name;
	}
}

JSP(javabean.jsp)

<%@ page import="lld.test.helloworld.PeopleBean" %>
...
<body>
	<jsp:useBean id="peopleBean" class="lld.test.helloworld.PeopleBean" scope="session" />
	<jsp:setProperty name="peopleBean" property="name" value="Peter" />
	<p>Hello, <jsp:getProperty name="peopleBean" property="name" /></p>
</body>
