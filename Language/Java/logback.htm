-Dlogging.config=my_logback.xml

初始化 Logger 对象

对于 static ，没办法只能 hard code 如下所示

static final Logger LOG = LoggerFactory.getLogger(MyClass.class);

如果不是 static，可以采用如下的简单写法

final Logger LOG = LoggerFactory.getLogger(getClass());
