排序

// List 排序
Collections.sort(testList);

// List 反序

工具类

Objects

Objects 提供了许多工具方法，例如比较字符串内容，获取 hash code 等，因为这些方法均支持 null 参数，所以可以简化我们的代码。

Objects.equals

String s1 = "hello";
Object s2 = "hello";

final boolean isEqual = Objects.equals(s1, s2);
System.out.println(isEqual);

Objects.hashCode

String s1 = "hello";
System.out.println(Objects.hashCode(s1));
