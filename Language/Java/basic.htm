Java语言基础

数据类型

基本类型(Primitive Data Types)和引用类型(Reference Types)。

基本类型包括：boolean, char, byte, short, int, long, float, double，其中int的字长为32位

引用类型为对象类型，即类，其中Object是所有类的基类，注意，基本类型的数组也是引用类型，如int[]也是Object的子类。

同步控制

使用synchronized关键字，如下所示

public static Sort(int[] a)
{
	synchronized(a)
	{
		//do the array sort here
	}
}

这样，我们就不用担心在排序的时候会有其它进程改变数组a的内容。因为a是按引用传递的，所以有可能在排序时，有其它进程改变数组a的内容。
