Maven 安装及配置

从 Apache Maven 官网下载后，设置如下环境变量

  • M2_HOME: Maven 安装目录
  • PATH: 将 %M2_HOME%\bin 加入 Path 环境变量
  • MAVEN_OPTS: -Xms128m -Xmx512m

在 Eclipse 中安装 maven 插件，并进行如下配置：

默认 Maven 会从默认的 central repository 下载类库。但如果外网连接不通而有相应的自己的 central repository，可通过修改 Maven conf 目录下的 settings.xml 文件，如下所示：

<mirror>
  <id>my_maven_repository</id>
  <mirrorOf>central</mirrorOf>
  <name>my maven repository</name>
  <url>http://myhost:8081/public/</url>
</mirror>

或者更省心，你把 mirrorOf 换成 * 就好

<mirrorOf>*</mirrorOf>

Build

第一个项目

在 Eclipse 随便建立一个 Hello World 项目，在根目录下添加 pom.xml，如下所示：

<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
  xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/maven-v4_0_0.xsd">
  <modelVersion>4.0.0</modelVersion>
  <groupId>com.lld</groupId>
  <artifactId>maven_test</artifactId>
  <packaging>jar</packaging>
  <version>1.0.0</version>
  <name>web Maven Webapp</name>
  <url>http://maven.apache.org</url>
  <build>
  	<sourceDirectory>src</sourceDirectory>
  </build>
</project>

常用命令

#编译
        mvn clean compile

        # 打包，默认为 jar 包
        mvn clean package

        # 生成默认的项目骨架 (pom.xml)
        mvn archetype:generate
