Servlet基本知识

需j2ee支持，如果不从java.sun.com下载，Tomcat下就自带，例如tomcat6下的lib目录下有servlet-api.jar，如需使用JSTL及标签支持，还需要使用jsp-api.jar

标准Servlet

最常用的使用方式为创建 javax.servlet.http.HttpServlet 的派生类，并重载其doGet和doPost方法。

获取页面传来的参数

使用对象javax.servlet.http.HttpServletRequest，是doGet或doPost方法的第一个参数，使用其 getParameter 方法获取页面传递的参数（GET方式和POST方式均可）

getParameter和getAttribute的区别

getParameter()是获取请求参数的值；getAttribute（）是获取保存在request|session|application中的一个名对应的对象

设置返回值

使用HttpServletRequest对象的setAttribute方法，设置JSP页面用到的服务器变量，在JSP中可以使用request.getAttribute获取。

跳转到返回页页

示例代码如下所示：

import javax.servlet.RequestDispatcher;
import javax.servlet.ServletContext;
...
ServletContext context = this.getServletContext();
RequestDispatcher dispatcher = context.getRequestDispatcher("/hello/servlet_jsp2.jsp");
dispatcher.forward(request, response);
