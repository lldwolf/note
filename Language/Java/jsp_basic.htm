JSP基本知识

N.1 基本用法

第一行的声明

<%@ page language="java" contentType="text/html; charset=GB18030" pageEncoding="GB18030"%>

在JSP中引用类

<%@ page import="java.util.*" %>

包含其它文件

<%@ include file="/inc/definition.inc"%>

在页面上输出

<%
	out.println("<br />");
%>

在页面上获取Attribute

<%
	String s = (String)request.getAttribute("xz");
%>

在页面上获取参数

<%
	String s = (String)request.getParameter("xz");
%>

N.2 常见错误

有时css或javascript在页面中可以正常使用，但放在单独的.css或.js文件中时便无法正常使用。这是由于文件编码方式引起的问题，可以使用记事本打开后，另存为UTF-8格式。
