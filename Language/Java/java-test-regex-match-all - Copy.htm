Java 练习题 - 调用 main 函数

要求

在 Java 中调用某个类的 main 函数

示例代码

public class A{
    public static void main(String[] args)
    {
        //Main code
    }
}

public class B{
    void someMethod()
    {
        String[] args = {"a", "b"};
        A.main(args);
        System.out.println("Back to someMethod()");
    }
}
