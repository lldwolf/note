Jar

Make Jar Package

jar -cf [jar-file-name.jar] [source-folder]

示例：把某个 jar 中的 class 单独打包为一个 jar

下面的例子将 spring-context.jar 中的 org.springframework.remoting.rmi.RmiInvocationWrapper_Stub 类单独打为 spring-2.5.6-rmi-compatibility.jar

jar -xvf spring-context.jar org/springframework/remoting/rmi/RmiInvocationWrapper_Stub.class
jar -cvf spring-2.5.6-rmi-compatibility.jar org/springframework/remoting/rmi

Run Jar

调用默认入口

java -jar [jar-file-name.jar]

调用指定类

java -cp [jar-file-name.jar] [full class name]
