Java 练习题 - String.format 拼接字符串

要求

使用 String.format 拼接字符串

示例代码

String str = String.format("Hello, %s", "Lindong");

使用的转换符

转换符 说明 示例
%s 字符串
%c 字符
%b 布尔
%d 整数(10进制)
%x 整数(16进制)
%o 整数(8进制)
%f 浮点数
%a 浮点数(16进制) FF.35AE
%e 科学计数法 9.38e+5
%g 通用浮点类型(%f 和 %e 中较短的)
%h 散列码
%% 打印百分号
%n 打印换行符
%tx 日期与时间为型, x 表示不同的日期与时间转换符

日期与时间的转换符 x 说明如下:

转换符 说明 示例
c 完整信息 星期六 十月 27 14:21:20 CST 2007
F 年-月-日 2017-2-28
D 月/日/年 2/28/2017
r HH:MM:SS PM (12 小时制) 02:25:51 PM
T HH:MM:SS (24 小时制) 14:25:51
R HH:MM (24 小时制) 14:25
b or h 月份简称 Oct
B 月份全称 October
a 星期简称 Sat
A 星期全称 Saturday
C 年的前两位数字 20
y 年的后两位数字 17
Y 4位数字的年份 2017
j 一年中的第几天
m 两位数字的月份 02
d 两位数字的日 09
e 月份的日 9
H 2位数的24小时制的小时(补0) 05
I 2位数的12小时制的小时(补0) 05
k 2位数的24小时制的小时(不补0)
l 2位数的12小时制的小时(不补0) 05
M 2位数分钟(补0)
S 2位数秒(补0)
L 3位数毫秒(补0)
p 小写字母上午/下午标记
z 相对于 GMT 的时区偏移量 +0800
Z 时区缩写字符串 CST
s 1970-1-1 到现在经过的秒数
S 1970-1-1 到现在经过的毫秒数
