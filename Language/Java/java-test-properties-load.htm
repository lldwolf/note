Java 练习题 - 加载 properties 文件

要求

给定 properties 文件的路径, 读取其内容

示例代码

Properties properties = new Properties();
		 
//使用 Class Path 相对路径加载
properties.load(AbstractTestCase.class.getClassLoader().getResourceAsStream("my.properties"));
		 
//使用绝对路径加载
properties.load(new FileInputStream("c:/temp/1.properties"));

//打印所有 property
Enumeration<?> enumeration = properties.propertyNames();
		
while (enumeration.hasMoreElements()) {
	String key = (String)enumeration.nextElement();
	String value = (String)properties.getProperty(key);
	System.out.println(String.format("%s=%s", key, value));
}
