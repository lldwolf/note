Eclipse 开发 Web 项目技巧

IDE 支持

打开 Servers Perferences，添加相应的 Tomcat 服务器，在 Tomcat 服务器节点上使用右键菜单 "Add and Remove ..." 可以将项目列表中的 Web 项目添加进 Tomcat webapps，然后右键启动 Tomcat 服务器即可

也可以在 Web 项目上使用右键菜单 "Run As" ⇒ "Run on Servers"。

FAQ

如何保证最新的代码被部署了

在 Servers Prefernece 中选中 Tomcat 服务器节点，先后使用右键菜单 "Clean" 和 "Publish" 即可.

如何查看真正的部署位置

在 Servers Prefernece 中选中 Tomcat 服务器节点下的 Web Application 节点，使用右键菜单 "Browse Deployment Location" 即可.
