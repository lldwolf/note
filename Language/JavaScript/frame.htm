控制FRAME

在frame子页面中，可使用 parent.frames["frame_name"].form1 或 window.frames["frame_name"].form1来访问其它子页面的元素

frame的属性与window类似，例如，可用如下语句使frame的页面跳转：

window.frames["frame_name"].location.href = "www.lld.com";

frame本身亦有document属性，如下语句可以取一个frame的内部元素：

var frame = document.frames["frame1"];
alert(frame.document.all["p1"].innerText);
