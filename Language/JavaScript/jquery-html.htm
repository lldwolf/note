使用 jQuery 控制HTML

jQuery 常用功能

//获取HTML对象
var divs = $('div');

//将HTML对象封装为 jQuery 对象
var divObj = $(divs[0]);

//查看 jQuery 对象的内部HTML内容 (innerHTML)
console.info(divObj.html());

//查看 jQuery 对象的完整HTML内容 (outerHTML)
console.info(($('<div>').append(divObj).clone()).html());

//或使用浏览器的新特性
console.info(divObj.prop('outerHTML'));

HTML 元素查找

操作 代码 说明
按 id 查找 
$('#myid')
 返回所有 id 为 'myid' 的元素
按标签查找 
$('div')
 返回所有的 div 元素
按 class 查找 
$('.myclass')
 返回所有 class 为 'myclass' 的元素
查找上一个同级元素 HTML 
<ul>
  <li>list item 1</li>
  <li>list item 2</li>
  <li class="third-item">list item 3</li>
  <li>list item 4</li>
  <li>list item 5</li>
</ul>
JavaScript 
$( "li.third-item" ).prev().css( "background-color", "red" );
 结果为 
<ul>
  <li>list item 1</li>
  <li style="background-color: red">list item 2</li>
  <li class="third-item">list item 3</li>
  <li>list item 4</li>
  <li>list item 5</li>
</ul>
查找下一个同级元素 HTML 
<ul>
  <li>list item 1</li>
  <li>list item 2</li>
  <li class="third-item">list item 3</li>
  <li>list item 4</li>
  <li>list item 5</li>
</ul>
JavaScript 
$( "li.third-item" ).next().css( "background-color", "red" );
 结果为 
<ul>
  <li>list item 1</li>
  <li>list item 2</li>
  <li class="third-item">list item 3</li>
  <li style="background-color: red">list item 4</li>
  <li>list item 5</li>
</ul>

HTML 结构编辑

可参考 http://api.jquery.com/

操作 代码 说明
添加 class 
$( "p" ).addClass( "myClass yourClass" );
 可添加多个, 使用空格格开
子元素封装 HTML 
<div class="inner">Hello</div>
JavaScript 
$( ".inner" ).wrapInner( "<div class='new'></div>");
 结果为 
<div class="inner">
	<div class="new">Hello</div>
</div>
父元素封装 HTML 
<div class="inner">Hello</div>
JavaScript 
$( ".inner" ).wrap( "<div class='new'></div>");
 结果为 
<div class="new">
	<div class="inner">Hello</div>
</div>
在开始处添加子元素 HTML 
<div class="inner">Hello</div>
JavaScript 
$( ".inner" ).prepend( "<p>Test</p>" );
 结果为 
<div class="inner">
	<p>Test</p>
	Hello
</div>
在结束处添加子元素 HTML 
<div class="inner">Hello</div>
JavaScript 
$( ".inner" ).append( "<p>Test</p>" );
 结果为 
<div class="inner">
	Hello
	<p>Test</p>
</div>
元素前插入同级元素 HTML 
<div class="inner">Hello</div>
JavaScript 
$( ".inner" ).before( "<p>Test</p>" );
 结果为 
<p>Test</p>
<div class="inner">Hello</div>
元素后插入同级元素 HTML 
<div class="inner">Hello</div>
JavaScript 
$( ".inner" ).after( "<p>Test</p>" );
 结果为 
<div class="inner">Hello</div>
<p>Test</p>
更改 inner text HTML 
<div class="demo-container">Hello</div>
JavaScript 
$( "div.demo-container" ).text( "Hello, Lindong" );
 结果为 
<div class="demo-container">Hello, Lindong</div>
更改 inner HTML HTML 
<div class="demo-container">
	<div class="demo-box">Demonstration Box</div>
</div>
JavaScript 
$( "div.demo-container" ).html( "<p>All new content. <em>You bet!</em></p>" );
 结果为 
<div class="demo-container">
	<p>All new content. <em>You bet!</em></p>
</div>
替换 HTML 元素 HTML 
<div class="container">
  <div class="inner first">Hello</div>
  <div class="inner second">And</div>
  <div class="inner third">Goodbye</div>
</div>
JavaScript 
$( "div.second" ).replaceWith( "<h2>New heading</h2>" );
 结果为 
<div class="container">
  <div class="inner first">Hello</div>
  <h2>New heading</h2>
  <div class="inner third">Goodbye</div>
</div>
