第三章 常用的JavaScript控制

对HTML元素的控制

控制标签是否可见

可使用如下代码：

--保留原占位空间
var tag = document.all["tag_name"];
tag.style.visibility = "visible";	//使可见
tag.style.visibility = "hidden";	//使隐藏

--不保留原占位空间
var tag = document.all["tag_name"];
tag.style.display = "block";		//使可见
tag.style.display = "none";		//使隐藏

其中 tag_name 为HTML标签的ID或Name属性，为了兼容性建议使用ID，因为有的浏览器不支持Name属性（IE支持）。

因为JavaScript为解释性语言，所以在所有的HTML标签装入前，上述代码不能起作用，例如直接在 <head> </head> 内嵌入上述代码，浏览器会报告错误，说找不到对象。
