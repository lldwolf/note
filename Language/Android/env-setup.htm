开发环境搭建

Android Studio

本来我是想用 Xamarine 用 C# 进行开发的。结果搭了半天环境也没搭起来，只好又回到了官方推荐的 Android Studio。

想从官网下载 Android Studio 并不容易，你得用各种姿抛翻墙。至少到 2017 年 5 月还是如此。我认怂，从百度软件下载了不那么新的版本。包括如下软件：

  • JDK 8
  • Android Studio 2.2

安装时注意路径中是否包含中文。我用的中文 Windows，用户名是中文，所以默认装到了 C:\Users\用户名\Android 目录下去了，然后踩入了一个大坑，不得不手工挪到了 C:\ 根目录下才避免了各种奇怪的路径问题。
