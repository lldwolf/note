Android Studio 使用

快捷键

快捷键 说明
Ctrl + Shift + 空格 智能自动提示，显示最可能的5个选项
Ctrl + Alt + 空格 自动补全代码
Alt + Insert 自动生成 Getter/Setter
Alt + F1 自动在项目树中定位到当前文件
Alt + ← / → 在文件 tab 中切换上一个/下一个
Ctrl + Alt + L 格式化代码
Ctrl + Alt + T 根据提示快速修复编译错误，例如加 try catch 块
Ctrl + O 重载父类方法
Shift + F10 启动程序
Ctrl + F2 停止运行
Ctrl + F6 重命名变量
