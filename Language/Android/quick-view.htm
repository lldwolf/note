学习过程

简要回顾

  1. 如何在 Android Studio 中新建项目
  2. 四大组件：Activity, Service, Broadcast Receiver, Content Provider
  3. 第一个程序，src 下的 java 目录和 res 目录
  4. 如何打印 log
  5. Log 过滤器
  6. 新建 Activity
  7. 简单布局，res 目录，LinearLayout
  8. 注册 Activity (AndroidManifest.xml)，添加 intent-filter 响应系统启动事件使之成为启动 Activity
  9. 为按钮注册事件
    • button.setOnClickListener
    • new View.OnClickListener
    • override onClick
  10. Toast 使用，Toast.makeText
  11. 使用 Menu，res 目录，新建 layout, activity override onCreateOptionsMenu, getMenuInflate().inflate()
  12. onOptionsItemSelected, 响应菜单事件, switch(item.getItemId)
  13. finish, 销毁 Activity
  14. Intent
    • 显式切换 Activity，startActivity(intent)
    • 隐式切换 Activity，AndroidManifest.xml, action, category, new Intent(action), intent.addCategory(category)
    • 打开浏览器，Intent.ACTION_VIEW
    • 注册针对 Intent.ACTION_VIEW 的 Activity, 可以响应系统浏览器请求
    • 打开拨号，Intent.ACTION_DIAL
    • 向下一个活动传递数据，intent.putExtra(key, value), getIntent().getStringExtra
    • 返回数据给上一个活动
      startActivityForResult(intent, request_code)
      intent.putExtra(), setResult(RESULT_OK, intent), finish
      override onActivityResult, switch(request_code), if (resultCode == RESULT_OK)
      override onBackPressed
      • Intent 生命周期，Create, Resume, Pause, Stop, Destroy, Restart
    • Activity 回收前保存临时状态 onSaveInstanceState
    • Activity 4种启动模式：standard, singleTop, singleTask, singleInstance
