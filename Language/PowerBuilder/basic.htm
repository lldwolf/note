N.1 新建一个Windows窗口程序

1、新建一个workspace，取名为Test1，PB生成Test1.pbw

2、new - Target - Application，取名为Test1，PB生成Test1.pbl（Library）和Test1.pbt（Target）。

3、new - PB Object - Window，取名win1，新建一个窗体

4、随便在窗体上放点啥

5、双击左侧导航栏内的Test1，即Library，在其open事件内写入代码如下：

 Open(win1)

6、Run - Run Test1，看运行结果。

N.2 将程序编译为EXE文件

在 N.1 节的基础上

1、New - Project - Application

2、在Execute File Name项中输入要生成的可执行文件名，其它的随便填

3、保存后，run - deploy wordspace，或直接按工具栏上的deploy按钮
