接口

接口就是最虚的类，只有方法，没有成员变量。在接口显可以定义义虚属性，如下所示：

public interface Interface1
{
	int Property
	{
		get;
		set;
	}
}

值得注意的是，如果使用Reflection方式来查看上面定义接口的成员，会发现有如下三个成员：

成员名 类型
Property Property
get_Property Method
set_Property Method

这意味着在后台会自动生成get_Property和set_Property函数，换言之，如果你再在interface中声明一个不带任何参数的get_Property()方法的话，会提示编译错误：

类型"Interface1"已定义了一个名为"get_Property"的具有相同参数类型的成员
