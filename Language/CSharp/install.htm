安装.net framework

命令行方式安装

在命令行方式下安装有如下参数：

/Q -- 无提示安装方式

/T:<完整路径> -- 指定临时工作文件夹

/C -- 与T并用，权将文件解开到指定文件夹

/C:<cmd> -- 忽略作者定义的安装命令，即解压后执行命令<cmd>

另有语法

dotnetfx [/q:a] [/c:"Install [/l][/q]"]

其中

选项 解释
/l 在%temp%目录中创建安装日志文件netfx.log，若安装出错则写入错误码
/q 在安装过程中不出现任何提示，也不会出现安装界面，必须同时指示选项/q:a

使用如下命令行安装则在安装过程中不出现任何提示和界面：

dotnetfx /q:a /c:"install /q"
