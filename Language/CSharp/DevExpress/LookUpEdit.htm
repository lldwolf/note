LookUpEdit

Basic Info

Skill

允许内嵌的 LookupEdit 输入 null (允许删除以前的选择)

LookupEdit Setup:

  • AllowNullInput: True
  • TextEditStyle: Standard

Necessary method & event:

  • LookupEdit.CloseUp Event
    • this.gridView1.ValidateEditor();
  • LookupEdit.ProcessNewValue Event
    • int rowHandle = gridView1.FocusedRowHandle;
GridColumn column = gridView1.FocusedColumn;
object value = (sender as LookUpEdit).Properties.GetKeyValueByDisplayText(e.DisplayValue.ToString());

this.BeginInvoke(new MethodInvoker(delegate
{
    gridView1.SetRowCellValue(rowHandle, column, value);
}));

e.Handled = true;

Note: The related field type must be nullable
