DevExpress.XtraTreeList.TreeList

使用 DataSource 属性显示数据时，TreeList会自动高亮 Id = 0 的节点。可通过重载TreeList控件的MakeNodeVisible方法避免。

public override int MakeNodeVisible(DevExpress.XtraTreeList.Nodes.TreeListNode node)
{
    if (this.FocusedRowIndex == -1 || this.FocusedNode == null)
        return -1;
    else
        return base.MakeNodeVisible(node);
}
