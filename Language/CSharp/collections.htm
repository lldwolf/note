集合类

概述

C# 提供了以下集合类型

  • 基本数组
  • System.Collections
    非范型集合类
  • System.Collections.Generic
    范型集合类
  • System.Collections.Concurrent
    线程安全集合类

常用接口

接口 说明
IEnumerable/IEnumerable<T> 支持枚举，即可用于 foreach 循环
ICollections 继承自 IEnumerable，动态集合，支持增删操作
IList/IList<T> 继承自 ICollections, 实现了索引器，可按下标索引，并实现了基于下标的增删操作
IDictionary<K, V> 哈希表

工具类

Enumerable 可用来初始化集合，如下所示

//a list with the initial value is 100, 10 elemens
List<int> list1 = Enumerable.Repeat(100, 10).ToList();

//a list with the initial value from 0 to 9
List<int> list2 = Enumerable.Range(0, 10).ToList();
