配置文件

针对Winform和ASP.NET有两种不同的配置文件，在Winform下为App.config文件，在ASP.NET中则为Web.config文件，对于这两个文件中的元素，.net提供了一组专门的类来进行操作。

App.config

appSettings部分

假设App.config的文件内容如下所示：

?xml version="1.0" encoding="utf-8" ?>
<configuration>
  <appSettings>
    <add key="Key1" value="Value1" />
    <add key="Key2" value="Value2" />
  </appSettings>
</configuration>

可使用如下代码来读取appSettings中的值

using System.Configuration;
...
Configuration config = ConfigurationManager.OpenExeConfiguration(null);
Console.WriteLine(config.AppSettings.Settings["Key1"].Value);
Console.WriteLine(config.AppSettings.Settings["Key2"].Value);
Console.Read();

Web.config

appSettings部分

如果在web.config中有如下配置片断：

appSettings>
	<add key="Key1" value="Value1" />
	<add key="Key2" value="Value2" />
</appSettings>

使用如下代码进行读取

using System.Web.Configuration;
...
string key1 = WebConfigurationManager.AppSettings["Key1"]
