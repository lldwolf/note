Dock的使用

使用Dock属性来停靠控件时，有时会出现这样的问题，例如，你先放一个Dock为Left的Panel，再放一个Dock为Top的Toolbar，你会发现Toolbar控件不能完全占据底部，而Panel控件完全占据了左部，出现如下图的效果：

如果想改成正常的模式，可将原先自动生成的代码：

this.Controls.Add(this.toolBar1);
this.Controls.Add(this.panel1);

改为：

this.Controls.Add(this.panel1);
this.Controls.Add(this.toolBar1);

这样显示就正常了，因为C#会用后面控件的Dock优先前高于前面控件。

Font的使用

修改字体的样式（斜体、粗体等） 

this.Font = new Font(this.Font, this.Font.FontStyle | FontStyle.Bold);		//设置粗体

修改字体的大小

this.Font = new Font(this.Font.FontFamily, 9, GraphicsUnit.Point);			//设置字体大小为9
