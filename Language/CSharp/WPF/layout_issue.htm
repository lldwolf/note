Status Bar

如何将其放在最下面

使用DockPanel, 将其放在DockPanel的第一个元素。如果不是第一个，效果为Fill

<DockPanel>
    <StatusBar DockPanel.Dock="Bottom">
        <StatusBarItem>
            <TextBlock Name="sbProject">Project:</TextBlock>
        </StatusBarItem>
    </StatusBar>
    ...
</DockPanel>

Style

类似于默认的 CSS，可实现指定控件的默认外观属性

<Window.Resources>
    <Style TargetType="TextBlock">
        <Setter Property="Margin" Value="3" />
    </Style>
</Window.Resources>

将此代码放到 Window 层，则将影响 Window 中所有的 textBlock，亦可将其放在任意控件层，例如放在 DockPanel 层，则将影响 DockPanel 内的所有 TextBlock.

但如果在控件中重新设置 Margin，则该控件将会使用自定义属性覆盖 Style 的设置。

亦可将该代码放到 App.xaml 中，则可应用于整个应用程序
