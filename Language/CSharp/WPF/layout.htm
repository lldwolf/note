常用布局

StackPanel

流式布局（横向或纵向）

Property Desc.
Orientation 控件排列方向
HorizontalAlignment 水平对齐方式

DockPanel

DockPanel 内的元素会被附加上 "DockPanel.Dock" 属性，用于指定元素将被 Dock 到哪个位置。没有 DockPanel.Dock 属性的元素将被认为将占据剩余空间，该元素应放在所有被 Dock 的元素后面。

Grid

类似于HTML中的Table布局方式。Grid 中分为三个区：

  • Grid.ColumnDefinitions: 内含 ColumnDefinition 元素，定义列
  • Grid.RowDefinitions: 内含 RowDefinition 元素，定义行
  • 元素区，元素将被附加上 Grid.Row 和 Grid.Column 属性，用于指示元素所在的行列。

ColumnDefinition

Width 属性用于设置列宽，可使用如下值

  • 80: 表示列宽为80px
  • Auto: 表示列宽将由控件的最大宽度决定
  • *: 表示将填满剩余空间
  • 80*: 表示列宽为80，若父控件（如窗体）尺寸改变，将同步改变

其它属性

Property Desc.
MinWidth 最小列宽，常在 Width 包含 “*” 的列中设定
MaxWidth 最大列宽，常在 Width 包含 “*” 的列中设定

Grid 布局示例

<Grid>
    <Grid.ColumnDefinitions>
        <ColumnDefinition Width="Auto" MinWidth="100" MaxWidth="150"></ColumnDefinition>
        <ColumnDefinition Width="*"></ColumnDefinition>
        <ColumnDefinition Width="80"></ColumnDefinition>
        <ColumnDefinition Width="4"></ColumnDefinition>
        <ColumnDefinition Width="80"></ColumnDefinition>
    </Grid.ColumnDefinitions>
    <Grid.RowDefinitions>
        <RowDefinition Height="25px"></RowDefinition>
        <RowDefinition Height="4"></RowDefinition>
        <RowDefinition Height="*"></RowDefinition>
        <RowDefinition Height="4"></RowDefinition>
        <RowDefinition Height="25"></RowDefinition>
    </Grid.RowDefinitions>
    <TextBlock Grid.Row="0" Grid.Column="0" VerticalAlignment="Center">Approver:</TextBlock>
    <TextBox Grid.Row="0" Grid.Column="1" Grid.ColumnSpan="4" Height="20" VerticalAlignment="Center"></TextBox>
    <TextBox Grid.Row="2" Grid.Column="0" Grid.ColumnSpan="5" BorderBrush="Black" AcceptsReturn="True"></TextBox>
    <Button Content="Submit" Grid.Column="2" Grid.Row="4"></Button>
    <Button Content="Cancel" Grid.Column="4" Grid.Row="4"></Button>
</Grid>

Canvas

类似于 WinForm 布局。控件会被附加上 Canvas.X 和 Canvas.Y 属性用于指定控件的位置。

布局技巧

Splitter

需基于 Grid 布局方式实现

<Window x:Class="WpfDemo.Layout.SplitterWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Title="SplitterWindow" Height="400" Width="616">
    <Grid>
		<Grid.ColumnDefinitions>
			<ColumnDefinition Width="150"></ColumnDefinition>
			<ColumnDefinition Width="Auto"></ColumnDefinition>
			<ColumnDefinition></ColumnDefinition>
		</Grid.ColumnDefinitions>
		<Grid.RowDefinitions>
			<RowDefinition Height="25"></RowDefinition>
			<RowDefinition></RowDefinition>
		</Grid.RowDefinitions>
		<TextBox Grid.ColumnSpan="3"></TextBox>
		<TextBox Grid.Row="1"></TextBox>
		<GridSplitter Grid.Row="1"
					  Grid.Column="1"
					  HorizontalAlignment="Center"
					  Width="5"
					  ShowsPreview="True"
					  VerticalAlignment="Stretch"></GridSplitter>
		<TextBox Grid.Row="1"
				 Grid.Column="2"></TextBox>
	</Grid>
</Window>
