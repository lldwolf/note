使用资源

常用资源示例

<ResourceDictionary 
  xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
  xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
  xmlns:sys="clr-namespace:System;assembly=mscorlib"
  xmlns:model="clr-namespace:WpfDemo.DataBinding.Models">

		<!-- String resource that can be localized -->
		<sys:String x:Key="localizedMessage">en-US Message</sys:String>

		<!-- 空字符串及字符串数组 -->
		<x:Array x:Key="arraySource"
				 Type="sys:String">
			<x:Static Member="sys:String.Empty" />
			<sys:String>Parent</sys:String>
			<sys:String>Child</sys:String>
		</x:Array>

		<!-- 本地对象（Data Islands） -->
		<model:Person x:Key="person" Name="Lindong" />

		<!-- 静态常量引用 -->
		<x:StaticExtension x:Key="static_constant" Member="model:Constants.CONST_STRING" />
</ResourceDictionary>

在 .NET Framework 4.5 后，默认用 ArrayExtension 来替代 Array.

对于上例中，我们使用了如下的一个本地类 WpfDemo.DataBinding.Models.Person，它包含一个　Name 属性

  1. 空字符串 
    <TextBlock Text="{x:Null}" />
  2. 特殊字符
    <TextBlock Text="A&#13;&#10;B" />

Resource 语法

       Resource 层次

Resource 可放置在任意的XAML层次中，例如，最外层为 Window，可将资源放入<Window.Resources>中

       资源文件拆分及合并

代码示例

<ResourceDictionary>
	<ResourceDictionary.MergedDictionaries>
		<ResourceDictionary Source="/Gollf.EMCT;component/Resources/ResourcesDictionary.xaml" />
		<ResourceDictionary Source="/Gollf.Infrastructure.ControlLibraryForWPF;component/Resources/ComponentResourceDictionary.xaml" />
	</ResourceDictionary.MergedDictionaries>
</ResourceDictionary>
