第N章 常用热键

快捷键 说明 备注
Shift + F6 编译当前项目 VS2005
Ctrl + Shift + B 编译  
F5 编译并运行  
F7 显示代码窗口  
Shift + F7 显示设计窗口，在aspx文件中可以切换HTML代码窗口与设计窗口  
F9 设置或取消断点  
Ctrl + Shift + A 添加新项  
Ctrl + KK 设置或取消书签  
Ctrl + KN 移到焦点到下一书签  
Ctrl + KP 移到焦点到上一书签  
Ctrl + F 查找  
Ctrl + H 替换  
Ctrl + N 新建  
Ctrl + RO 重定义参数顺序 VS2005
Ctrl + RR 重命名变量 VS2005
Ctrl + EW 文本自动换行  
Ctrl + U 选中文本变为小写  
Ctrl + WL 服务器资源管理器
