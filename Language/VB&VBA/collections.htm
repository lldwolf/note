第N章 使用动态对象数组

如果你已经使用过Java或.net之类的语言，应该对ArrayList这种动态对象数组不陌生。所幸，在VBA中也有类似的对象Collection。使用方法如下：

先建立一个类模块Class1，它的内容很简单，只有两个公用变量：

Public Attr1 As Integer
Public Attr2 As String

然后建立测试模块，内容如下：

Public Sub test()

    Dim t As Class1
    
    Set t = New Class1
    
    t.Attr1 = 10
    t.Attr2 = "safdsaf"
    
    Dim a As Collection
    Set a = New Collection
    
    Call a.Add(t)
    
    Dim b As Class1
    
    Set b = a(1)
    
    MsgBox b.Attr2

End Sub
