N.1 基于表格

N.1.10 常见错误

表格不能填充整个页面

表现形式：

表格的样式如下，三行，上下两行在页面两端，中间行填充剩余区域。

<table width="100%" style="height: 100%" border="1">
	<tr style="height:40px" valign="top">
		<td>&nbsp;</td>
	</tr>
	<tr>
		<td>&nbsp;</td>
	</tr>
	<tr style="height:40px" valign="bottom">
		<td>&nbsp;</td>
	</tr>
</table>

正常情况下应该是填满整个页面的，但如果页面使用了XHTML 1.0规范，则在某些浏览器上（例如Firefox）不能正常显示。使用该规范时，页面开始的代码如下所示：

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">

将该段代码删除，直接使用<html>标签，则可解决此问题，此时默认使用的规范是HTML 4。
