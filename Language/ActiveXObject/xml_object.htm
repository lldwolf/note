第N章 XML

N.1 概述

主要使用的对象有如下几个

对象名 描述
MSXML2.DOMDocument XML文档对象，类似于.net中的System.XML.XmlDocument
MSXML2.IXMLDOMElement XML结点，类似于.net中的System.XML.XmlElement
MSXML2.IXMLDOMAttribute 结点属性，类似于.net中的System.XML.XmlAttribute
MSXML2.IXMLDOMNodeList MSXML2.IXMLDOMNode的集合
