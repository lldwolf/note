快捷键

快捷键 说明
Win+E 打开文件管理器
F2 修改文件名
F6 在文件管理器各个面板间移动
F4 进入文件管理器地址栏并显示最近使用的地址列表
Alt + ↑ 进入父文件夹
Ctrl + N 打开一个与当前位置相同的新的文件管理窗口
Alt + D 进入文件管理器地址栏
Ctrl + W 关闭文件管理器
F11 切换文件管理器的全屏模式
Alt + Enter 显示选中文件的属性
Ctrl + Shirt + N 新建文件夹
Win+↑ 最大化窗口
Win+↓ 恢复/最小化窗口
Win+←/→ 在双屏环境下将窗口在双屏间移动
Win+D 显示桌面
Win+L 锁屏
Win+R 打开运行命令窗口
Win+B 进入任务托盘
Win+T 进入任务栏，然后可以用光标键切换
Win+1...9 打开任务栏上的指定窗口
Alt+Shift+Tab 反向切换窗口
Ctrl+Shift+Esc 打开任务管理器
Alt + Shift 输入法开关
Ctrl + Shift 切换输入法
PrtSc/Alt + PrtSc 截屏/截当前窗口
