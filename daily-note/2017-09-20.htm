I/O & Linux Shell

今天使用 Java 动态生成一个 Linux Shell 脚本并执行，总是报 Permission Denied 错误。后来发现是生成的文件没有执行权限，那么生成文件后调用 java.io.File.setExecutable 就好了。

另外，可以使用 chmod +x file 来给文件赋执行权限。
