jQuery

在使用 jQuery 时犯了个错误，把访问 DOM 组件的代码写 DOM 组件前面了

<script>
	$('#btn').click(...);
</script>
...
<button id="btn" ... />

在使用如下方式使用 jQuery ajax post 时，总是会提交 form，使用了如下方式避免

<script>
	$('#btn').click(function(e) {
        e.preventDefault();
		$.post(...);
	});
</script>

jQuery 的 ajax 使用语法及示例

var url = 'my.do';	//要提示的目标地址
var pars = { action: "refresh" }; //以 json 方式传递参数，后端可以使用 request.getParamer("action") 得到
var callback = function(data) { alert data; }; //请求成功后的回调函数，data 可由后端的 response.getOutputStream().print() 写入
$.post(url, pars, callback);
