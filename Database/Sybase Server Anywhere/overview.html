第一章 概述

N.1 启动

我使用的版本为7.0，假如有一个数据库文件c:\demo.db，创建该文件的数据库服务的命令如下：

dbsrv7 -n demo -c 8m -x tcpip "c:/demo.db"

其中dbsrv7.exe存在于Adaptive Server Anywhere的安装目录下，例如，我本机的安装位置为 D:\Program Files\Sybase\SQL Anywhere 7\win32

上述命令中，其中 -n 参数指定启动后的服务名，-c 参数指定初始的内存大小，-x 参数指定使用的通信协议，最后是要打开的数据库文件路径。

标准帮助文件如下所示：

Usage: dbsrv7 [server switches] {database [switches]}*
       @data expands "data" from environment variable "data" or file "data"

Server Switches(apply to server and all databases):



	

		Switch
		Description

		-ca 0
