字段类型

N.1 自动编号

其SQL类型为counter或autoincrement，可带两个参数，分别为初始值和自增量。例如：

create table test1([id] autoincrement primary key)
create table test2([id] autoincrement(200, 10) primary key)
alter table test1 alter column [id] counter(100, 5)
