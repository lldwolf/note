第N章 系统对象

N.1 获取数据库中所有的表

SELECT *
FROM MSysObjects
WHERE [Flags]=0 AND [Type]=1

当我读取ACCESS里的系统表MSysObjects时，出现：不能读取记录；在 'MSysObjects' 上没有读取数据权限。的提示，而这个语句是可以直接在ACCESS查询器里使用的，不知为何，网游一翻，找到解决方法，不过还好，按这样的设置后以后这个数据库就可以读取这个表了。

操作过程：

默认的情况是admin对MSysObjects没有任何权限！包括查询

打开您的ACCESS数据库，然后

工具 -> 选项 -> 视图 -> 隐藏对象、系统对象
前面的勾选上。

工具 -> 安全 -> 用户与组的权限
在对象名称中选定 MSysObjects ，然后权限中设置其权限。

OK，现在可以试一试了，以后这个数据库就可以直接读取MSysObjects表了。
