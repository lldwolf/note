权限管理

设定对表的访问权限

表的权限有如下几种：

权限名 说明
ALTER  
DELETE  
INDEX  
INSERT  
REFERENCES  
SELECT  
UPDATE  

赋权限的命令为grant，格式如下：

grant 权限名 on 用户名1.表名 to 用户名2

例如把用户user1的表table1的select权限赋给用户user2，SQL语句如下：

grant select on user1.table1 to user2

如果使user2对user1.table1有全部权限，可用如下语句

grant all privilege on user1.table1 to user2

SYSDBA权限

虽然可将DBA权限赋给用户，但实际上只有SYS用户拥有DBA权限并可使用SYSDBA身份登录，其实在WINDOWS下，你会发现你随便输入用户名和口令都可以以SYSDBA身份登录，这是因为在安装ORACLE时，默认地把Administrator操作系统用户加到了ORA_DBA组中了，只要把该组中的Administrator用户删掉就可以限制在使用SYSDBA身份登录时只能用SYS用户了。
