N.2 执行动态SQL语句

类似于SQL SERVER 中 sp_executesql 存储过程的功能。可以动态地组成SQL字符串并执行，使用该功能，除了使得PL/SQL程序可以更灵活外，还可以避开一些限制。例如，在存储过程中是不能创建表的，但使用DBMS_SQL则可避开这一限制。

下面是一个使用DBMS_SQL执行动态SQL语句的简单例子：

declare 
  i integer;
  v_cursor_id integer;
  v_update_stmt varchar2(1000);
  v_updated_rows int;
begin
  v_cursor_id := DBMS_SQL.open_cursor;
  v_update_stmt := 'create table test_table2(aaa int)';
  DBMS_SQL.parse(v_cursor_id, v_update_stmt, DBMS_SQL.v7);
  v_updated_rows := DBMS_SQL.execute(v_cursor_id);
  DBMS_SQL.close_cursor(v_cursor_id);
end;
