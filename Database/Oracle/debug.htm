使用DBMS_OUTPUT.put_line()

这个命令并不是我们平常想象的那样，即时打印消息的。而是打印到内部缓冲区中，缓冲区的默认大小是20,000字节，最大大小是1,000,000字节，如果超出将会提示溢出错误，可以通出DBMS_OUTPUT.ENABLE和DBMS_OUTPUT.DISABLE命令启用或禁用缓冲区，并设置缓冲区大小。缺省是EABLE，如果调用了DISABLE，则缓冲区中的内容将会被冲掉，以后再调用PUT_LINE将没有任何影响。

ENABLE语法

PROCEDURE ENABLE(buffer_size IN INTEGER DEFAULT 20000);

PROCEDURE DISABLE;
