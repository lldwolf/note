一个常用的SQL语句，用来查询哪些表具有某个列名：

SELECT * FROM SYS.OBJ$ 
WHERE OWNER# = 64 AND TYPE# = 2 AND OBJ# IN (SELECT OBJ# FROM SYS.COL$ WHERE NAME = 'REV_ID')
ORDER BY NAME
