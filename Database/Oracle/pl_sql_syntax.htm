第N章 语法

N.1 表达式

N.1.1 赋值表达式

  • 使用赋值运算符 
    variable := expression
  • 使用SELECT语句 
    select column_list into variable_list or record_variable 
from ...
    可使用与所选列结构相同的记录类型

N.1.2 算术运算符

+、-、*、/

**：指数运算符

N.1.3 关系运算符

>、<、=、<=、>=

<>、!=、~=、^=均表示不等于

对于等于关系算符在判断字符串是否相等时，分为定长比较(blank-padded)和非定长比较(non-blank-padded)，对于定长比较，会通过在长度较小的字符串两端加空格补齐长度的方式来比较。关键是看使用的变量类型，char使用定长方式比较，varchar2使用变长方式比较，常量使用定长方式比较。

like为比较运算符，其中“%”通配多个字符，“_”通配单个字符。

between...and...相当于<=和>=的组合。

N.1.4 逻辑运算符

逻辑运算符：AND、OR、NOT

N.2 控制结构

N.2.1 if...then...else

IF boolean_expression1 THEN
	...
ELSIF boolean_expression2 THEN
	...
ELSE
	...
END IF;

在PL/SQL中没有代码块的分界符（类似于C中的{}）。

注意elsif的拼写，没有e。

N.2.2 loop 循环

LOOP
	...
	EXIT [WHEN contintion];
END LOOP;

N.2.3 while 循环

WHILE condition LOOP
	...
END LOOP;

N.2.4 for 循环

FOR loop_counter IN [REVERSE] low_bound...high_bound LOOP
	...
END LOOP;

如果使用REVERSE关键字，表示循环索引将从大到小递减。

N.2.5 goto

GOTO label;

标签需要用<<>>包括起来，如下例所示：

goto lable1;
...
<<label1>>

N.2.6 NULL 语句

占位符，啥也不做，类似于C语言中的空语句。
