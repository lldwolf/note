第N章 系统参数

系统参数 简要说明
nocount 使返回的结果中不包含有关受 Transact-SQL 语句影响的行数的信息。
identity_insert 可否插入identity列（标识列）
   
   
   
   
   
   
   
   

nocount

在执行SQL语句时，例如insert，系统会返回有多少行受到了影响，该参数即关闭该信息。

identity_insert

该参数确定是能否手工插入identity列。

语法：

set identity_insert 表名 on
--insert 语句
set identity_insert 表名 off
