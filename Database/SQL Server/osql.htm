我不知道先有的OSQL还是先有的SQL*PLUS，明摆着都是一路货色，今天不经意发现SQL SERVER还有这么个东东

[-S server] [-H hostname] [-E trusted connection]
[-d use database name] [-l login timeout] [-t query timeout]
[-h headers] [-s colseparator] [-w columnwidth]
[-a packetsize] [-e echo input] [-I Enable Quoted Identifiers]
[-L list servers] [-c cmdend] [-D ODBC DSN name]
[-q "cmdline query"] [-Q "cmdline query" and exit]
[-n remove numbering] [-m errorlevel]
[-r msgs to stderr] [-V severitylevel]
[-i inputfile] [-o outputfile]
[-p print statistics] [-b On error batch abort]
[-O use Old ISQL behavior disables the following]

[-? show syntax summary]

如下代码使用SA登录本机

osql /U sa /P password

然出就出现提示符，然后你就敲SQL语句吧，只不过与sql*plus的以字符“;”结束命令不同，它是使用“go”作为SQL语句的结束

ISQL和OSQL是一样的东东，对用户而言没有什么区别，只不是OSQL使用的是ODBC与服务器通讯，ISQL使用的是DB-Library。
