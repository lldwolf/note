查询分析器使用技巧

快捷键

快捷键 说明
ALT + - 子窗口的系统菜单，常用于最大化子窗口
CTRL + R 显示或关闭结果窗口
CTRL + D 以表格方式显示结果
CTRL + T 以文本方式显示结果
CTRL + B 调整结果窗口的大小
CTRL + F5 语法检查
F5 运行
F8 显示或关闭数据库导航窗口
CTRL + N 新建窗口
CTRL + O 新建连接
CTRL + SHIFT + P 打开SQL文件
   

结果字符集

对于非中文字符集的数据库，有时里面也可以存储中文，但是是以扩展ASCII码的形式存放，例如，“中文”可能被存为内码字符“ÖÐÎÄ”。如果要在结果集中显示对应中文，可取消“工具” - “选项” 窗口中“连接” 面板内的“执行字符数据的转换” 检查框。
